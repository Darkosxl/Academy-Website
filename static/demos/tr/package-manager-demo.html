💻 Projeniz

my-app/ ├── index.js ├── package.json └── node_modules/ (boş)

1 paket talep ettiniz. express tıklayın ve gerçekte nelerin geldiğini görün.

Bir paketi tıklayarak kendi bağımlılıklarını genişletin. İşte bu yüzden node_modules ağırdır — ve paket yöneticilerinin var olma nedeni budur: kimse bunu elle takip edemez.

Her dil kendi paket yöneticisini geliştirdi. Her yerde iş aynı: paketleri indir, sürümleri çözümle, kilit dosyasında takip et.

📦 npm

JavaScript / Node.js

Varsayılan seçenektir — Node ile birlikte gelir. Dünyanın en büyük kayıt defteri (~3 milyon paket). JS dünyasındaki diğer her şey buna bir tepki niteliğindedir.

npm install express

⚡ pnpm

JavaScript / Node.js

npm'nin her projeye aynı dosyaları kopyalaması nedeniyle ortaya çıkmıştır. pnpm tek bir global kopya saklar ve buna sabit bağlantı kurar — daha hızlı yükleme, büyük disk tasarrufu sağlar.

pnpm add express

🧶 Yarn

JavaScript / Node.js

npm'nin yavaş ve belirsiz olduğu dönemde Facebook'ta ortaya çıkmıştır. npm'nin daha sonra benimsediği kilit dosyası fikrini icat etmiştir. Eski kod tabanlarında hâlâ yaygındır.

yarn add express

🐍 pip

Python

Python'un standart yükleyicisi, PyPI'dan indirir. Genellikle sanal ortamlarla birlikte kullanılır, böylece projeler sürümler konusunda çatışmaz.

pip install requests

🦀 Cargo

Rust

Genel olarak en iyi tasarlanmış olanı olarak kabul edilir: tek bir tutarlı komutta paket yöneticisi, derleme aracı ve test çalıştırıcısı bir arada bulunur.

cargo add serde

🍺 Homebrew

macOS / Linux sistem uygulamaları

Aynı fikir, farklı hedef: bir projeye kütüphaneler yüklemek yerine, makinenize programların tamamını yükler.

brew install ffmpeg

💎 Bundler / gem

Ruby

Ruby'nin eşi: gem yükler, Bundler ise proje başına tam sürümleri sabitler. Gemfile.lock dosyası, kilit dosyalarını her yerde yaygınlaştırdı.

bundle add rails

☕ Maven / Gradle

Java / JVM

Kurumsal ağır toplar — bağımlılık yönetimi ve eksiksiz bir derleme boru hattı. Gradle, daha yeni, daha hızlı ve daha az XML kullanan seçenektir.

gradle build
