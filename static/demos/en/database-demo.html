What Is a Database?

Not magic — just organized tables you can query. Click around.

A database is a grid of grids

At its simplest, a database is a set of tables — each one like a spreadsheet with strict rules:

  • Each column holds one type of thing (name, price, date).
  • Each row is one record (one user, one order).
  • One column is usually the primary key — a unique ID for that row.
  • Tables link to each other via foreign keys — an ID that points to a row in another table.
